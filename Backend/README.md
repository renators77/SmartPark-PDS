## Description

Backend da app SmartPark

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode(Usar este modo para ativar o nodemon)
$ npm run start:dev

#correr os testes
$ npm run test:watch

# production mode
$ npm run start:prod

# criar module
$ npx @nestjs/cli g module permissao

# criar controler
$ npx @nestjs/cli g controller permissao

# criar service
$ npx @nestjs/cli g service permissao
```
